# Contributing to Alf Agent

**Alf Agent** is a **fork** of **[Pi](https://pi.dev)** (Pi Agent) by **[Mario Zechner](https://mariozechner.at)** ([@badlogic](https://github.com/badlogic)); upstream is **[earendil-works/pi-mono](https://github.com/earendil-works/pi-mono)**.

## Governance

This fork is maintained **BDFL-style** by **[@dpopsuev](https://github.com/dpopsuev)**. Direction and merges are not consensus-driven.

## Contributions

**Outside contributions are not accepted.** Do not open pull requests or issues expecting review, discussion, or merge. They may be closed without reply.

The project is **open to read** and **open to fork** under the **MIT** license: clone it, adapt it, and ship your own fork.

Improvements that belong in the original Pi line should go **upstream** to **[earendil-works/pi-mono](https://github.com/earendil-works/pi-mono)**.

## Agents

If you use coding agents against this tree, run them from the repository root so they pick up **`AGENTS.md`**.
