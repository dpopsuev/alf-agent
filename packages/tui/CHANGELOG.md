# Changelog

All notable changes to **`@dpopsuev/alef-tui`** are recorded here.

Release history before **[v0.0.1]** belongs to upstream **[Pi](https://github.com/earendil-works/pi-mono)** (`earendil-works/pi-mono`).

## [0.0.1] - 2026-05-10

### Changed

- Initial **`@dpopsuev/alef-tui`** release from the **[alef](https://github.com/dpopsuev/alef)** fork baseline.
- Editor and autocomplete now treat **`:`** as the operator-command surface and reserve **`/`** for filesystem paths, aligning TUI completion with Alef's symbolic discourse grammar.
